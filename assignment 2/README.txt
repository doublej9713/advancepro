1. To run the code, tesseract.exe needed to download from link: https://github.com/UB-Mannheim/tesseract/wiki.
2. Prepare the pytesseract and opencv library in PyCharm by using the command line :
   	“pip install pytesseract”
	“pip install opencv-python”
3. Declare the path of tesseract.exe inside the code
4. Prepare a sample image for the project and declare the path of the sample image inside the code.
5. Run the python code.
